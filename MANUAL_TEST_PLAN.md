# Σχέδιο Ελέγχου - Manual Test Plan
## TeacherVionlin Application - Έλεγχος Βασικών Λειτουργιών CRUD

### 📋 Γενικές Πληροφορίες

**Σκοπός:** Επαλήθευση ότι όλες οι βασικές λειτουργίες της εφαρμογής (CRUD) λειτουργούν σωστά μετά τις πρόσφατες διορθώσεις.

**Ημερομηνία Δημιουργίας:** 28 Οκτωβρίου 2025

**Προαπαιτούμενα:**
- Η εφαρμογή να είναι εγκατεστημένη και να τρέχει
- Σύνδεση με έγκυρο λογαριασμό χρήστη
- Σύνδεση με τη βάση δεδομένων Supabase
- Πρόσβαση στο ημερολόγιο της εφαρμογής

**Οδηγίες Εκτέλεσης:**
1. Εκτελέστε κάθε test case με τη σειρά που παρουσιάζεται
2. Ακολουθήστε τα βήματα ελέγχου ακριβώς όπως περιγράφονται
3. Επαληθεύστε ότι το αποτέλεσμα ταιριάζει με το αναμενόμενο
4. Σημειώστε "PASSED" αν το test πέρασε επιτυχώς
5. Σημειώστε "FAILED" αν το test απέτυχε, με σύντομη περιγραφή του προβλήματος
6. Επιστρέψτε τον πίνακα συμπληρωμένο για αξιολόγηση
